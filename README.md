# 🎯 Day 1 – Guess the Number Game

## 🎮 Description

A simple game where the player has to guess a number between 1 and 10.

---

## ⚙️ Features

* 3 chances
* Hints (Too high / Too low)
* Win/Lose result

---

## ▶️ How to Run

```bash
python guess_game.py
```

---

## 🧠 What I Learned

* Basic Python input/output
* Conditional statements
* Loops and game logic
